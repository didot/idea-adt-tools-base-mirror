This is a sample client running on Android devcie to illustrate how the server behaves.

### How to build?

1. Intall Android NDK.
2. Type 'ndk-build' in this directory ("tools/base/profiler/profiling_server/sample_client_on_device").

### How to deploy?

adb push libs/armeabi/sample_client_on_device /data/local/tmp

### How to run?

Run the binary directly from adb shell.
